## About this repository
an app for using build_system_kit



## How to use?

	$cd ffmpeg_audio
    $./build_lib.sh
    
    $make menuconfig
    $source run_make.sh
    $source set_ffmpeg_environment.sh

    $./build/ffmpeg_audio Despacito-16666406.mp3
